## What The Program Does

Bastions Dayz Tools gives DayZ server owners a clean visual workspace for common configuration jobs:

- Edit loot economy files such as `types.xml`
- Build starting loadouts visually
- Edit server settings through guided panels
- Work with DayZ Expansion traders, markets, and settings
- Manage supported community mod configuration files
- View logs and work with local server/profile folders
- Sync server files through FTP/SFTP workflows where supported

## Install Steps

1. Download the latest setup file above.
2. Run `Bastions-Dayz-Tools-Setup-1.3.7.exe`.
3. Open Bastions Dayz Tools.
4. Choose or create your DayZ server workspace.
5. Start editing your configuration files from the app panels.

## Windows SmartScreen

Because this installer is new and not yet code-signed, Windows may show a SmartScreen warning.

If prompted:

1. Click **More info**.
2. Click **Run anyway**.
3. Continue installation normally.
